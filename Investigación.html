Investigación Fase 2

  • Etiqueta Body

    • Este define el contenido principal de una pagina web HTML, incluyendo textos, imágenes, tablas etc.

  • Etiqueta Title

    • Define el titulo de la página web y es obligatioria en los documentos HTML

  • Etiqueta p

    • Define un parrafo o un bloque de texto nuevo

  • Etiqueta Table

    • Define y crea elementos de una tabla

  • Etiqueta tr

    • Define la fila de una tabla

  • Etiqueta td

    • Define celdas de datos en una tabla

  • Etiqueta th

    • Define las celdas de un encabezado de la tabla

  • Etiqueta Ol

    • Define una lista ordenada, es decir que todos los elementos están ordenadas de manera irracional

  • Etiqueta Li

    • Define los elementos de una fila de manera desordenada

  • Etiqueta form

    • Define un formulario, una sección en el que el usuario da sus datos

  • Etiqueta Input

    • Se utiliza para crear controles interactivos para el usuario

  • Etiqueta label

    • Añade o crea una leyenda etiqueta explicativa

  • Etiqueta select

    • Crea una lista desplegable para que los usuarios tengan varias opciones
